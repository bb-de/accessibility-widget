Barrierefreiheit-Widget Implementierung

Auf dieser Seite wird gezeigt, wie einfach das Barrierefreiheit-Widget auf Ihrer Webseite implementiert werden kann.

So einfach ist die Installation

Fügen Sie dieses Script-Tag direkt vor dem schließenden </body>-Tag Ihrer Webseite ein:

<script 
  src="https://your-widget-url.vercel.app/widget-loader.js" 
  data-token="YOUR_TOKEN" 
  data-position="right" 
  data-lang="de"
  async
></script>

Hinweis: Ersetzen Sie YOUR_TOKEN durch Ihren persönlichen Zugriffstoken und your-widget-url.vercel.app durch die tatsächliche URL, unter der das Widget gehostet wird.

Konfigurationsoptionen

Das Widget kann durch verschiedene Datenattribute angepasst werden:

  • data-token: Ihr eindeutiger Zugriffstoken (erforderlich)
  • data-position: Position des Widget-Buttons ("left" oder "right", Standard: "right")
  • data-lang: Sprache der Widget-Oberfläche ("de", "en", "fr", "es", Standard: "de")
  • data-theme: Design-Variante ("default", "dark", "light", Standard: "default")

Demo des Widgets

Klicke auf den Button, um das Barrierefreiheit-Widget zu öffnen und die verschiedenen Funktionen zu testen.

Vorteile des Widgets

  • Einfache Integration mit nur einem Script-Tag
  • Keine Beeinträchtigung der Ladegeschwindigkeit dank asynchronem Laden
  • Umfassende Barrierefreiheits-Funktionen für verschiedene Benutzergruppen
  • Mehrsprachige Unterstützung
  • Anpassbares Design
  • Anonyme Nutzungsstatistiken für bessere Optimierung
