Barrierefreiheits-Widget für GitHub Pages

Diese Beispielseite zeigt, wie das Barrierefreiheits-Widget über GitHub Pages eingebunden wird.

So bindest du das Widget ein:

Füge einfach folgenden Code-Schnipsel vor dem schließenden </body>-Tag deiner Website ein:

<script src="https://bb-de.github.io/accessibility-widget/github-widget-loader.js" data-position="right" data-lang="de" data-theme="default" data-debug="true"> </script>

Konfigurationsoptionen

Das Widget kann über folgende Attribute angepasst werden:

  • data-position: Position des Widget-Buttons ("right" oder "left")
  • data-lang: Sprache des Widgets (de, en, fr, es)
  • data-theme: Farbschema des Widgets (default, dark, light, high-contrast)
  • data-debug: Debug-Modus aktivieren (true/false)

Hinweis zur Content Security Policy (CSP)

Wenn deine Website eine strikte Content Security Policy verwendet, musst du möglicherweise folgende Direktiven hinzufügen:

Content-Security-Policy: script-src 'self' https://bb-de.github.io; style-src 'self' https://bb-de.github.io 'unsafe-inline'; connect-src 'self' https://bb-de.github.io; img-src 'self' https://bb-de.github.io; font-src 'self' https://bb-de.github.io;
