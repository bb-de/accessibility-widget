Barrierefreiheits-Widget Einbindung

Diese Beispielseite zeigt, wie das Barrierefreiheits-Widget in jede Website eingebunden werden kann.

So bindest du das Widget ein:

Füge einfach folgenden Code-Schnipsel in den <head> oder <body> deiner Website ein:

<script src="https://bb-de.github.io/accessibility-widget/accessibility-widget.js"></script>

Konfigurationsoptionen

Das Widget kann über folgende Attribute angepasst werden:

  • position: Position des Widget-Buttons ("right" oder "left")
  • language: Sprache des Widgets (de, en)
  • theme: Farbschema des Widgets (default)

Beispiel für angepasste Konfiguration:

<script> // Widget-Konfiguration anpassen window.onload = function() { if (window.AccessibilityWidget) { // Sprache auf Englisch ändern window.AccessibilityWidget.setLanguage('en'); // Ein Profil direkt anwenden window.AccessibilityWidget.applyProfile('visionImpaired'); } }; </script>

Hinweis zur Content Security Policy (CSP)

Wenn deine Website eine strikte Content Security Policy verwendet, musst du möglicherweise folgende Direktiven hinzufügen:

Content-Security-Policy: script-src 'self' https://bb-de.github.io; style-src 'self' https://bb-de.github.io 'unsafe-inline';
